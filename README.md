smtp-sender
===========

SMTP로 메일을 전송하는 Java GUI 프로그램

![smtp-sender v0.1.0](http://www.inter6.com/_media/mail:smtp-sender_v0.1.0.png)

v0.1.0 (Current)
- 인증없는 SMTP 전송
- MIME 텍스트, EML 파일 전송

v0.2.0
- Mail From, Rcpt To를 헤더값으로 전송
- 디렉토리 내의 모든 EML 전송
- 멀티 쓰레드 지원
- 로깅 패널 개선
